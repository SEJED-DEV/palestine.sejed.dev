# Contributing to Palestine Educational Site

Thank you for considering contributing. This project aims to provide accurate, source-backed educational content about Palestine in 4 languages.

## How to Contribute

### Content & Translations
- Add or correct translation keys in `messages/{locale}.json`
- Ensure every factual statement has a verifiable source
- All content must be respectful and historically accurate

### Code
- Fork the repo and create a feature branch
- Follow the existing code style (no comments, inline styles, functional components)
- Run `npm run lint` before submitting
- Open a pull request with a clear description

### Bug Reports & Ideas
- Open an issue on GitHub
- Include screenshots if relevant
- Describe expected vs actual behavior

## Code of Conduct
Be respectful. This project is educational and solidarity-based. No hate speech, no harassment, no spam.
